## Drag and Drop

### Load NPM pachages
```
npm i
```

### Run BE server
```
npm run server:start
```

### Run FE server
```
npm start
```

### Run StoryBook 
```
npm run storybook
```

### Run Jest Tests
```
npm run test
```

### Run Cypress Test
Make sure you have your FE and BE running before running this test.
```
npm run cy:open
```
